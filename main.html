The internet, as we know it, has passed two major updates...

Web 1.0
1989 - 1999

Websites used to be static, this means, there was no interaction oportunities for the user. An easy way to visualize this is to think in a blog, but with no chance to put comments.

Web 2.0
1999 - At present

Users became able to publish articles, comments, create accounts and so on. This era saw the begining of giant apps like Facebook, Twitter, Wikipedia, etc.
